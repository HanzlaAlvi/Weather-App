WeatherVision Pro+ 🌦️
A Modern Python Weather Application with GUI

WeatherVision Pro+ Screenshot

Features
    ✔ Real-time Weather Data (Current + 5-Day Forecast)
    ✔ Beautiful Tkinter GUI with Light/Dark Themes
    ✔ Voice Weather Reports (Text-to-Speech)
    ✔ Weather Alerts (Extreme Temp, Storms, etc.)
    ✔ Email Weather Reports
    ✔ Historical Data Graphs
    ✔ Automatic Icon & Data Caching

Tech Stack
    Python 3 + Tkinter (GUI)
    OpenWeatherMap API (Weather Data)
    Pandas (Data Handling)
    Matplotlib (Graphs)
    gTTS (Voice Reports)

How to Use
    Search for any city worldwide.
    Toggle between °C/°F and Light/Dark Mode.
    Click "🔊" for voice forecasts.
    Export data via Email or Excel.
